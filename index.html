Welcome to ACL AFS

This application allows to apply recursively (or not) OpenAFS ACLs in Windows. It is mainly a frontend for fs.exe.

Install

Go to https://github.com/falsovsky/ACLAFS/releases and download the latest release there (aclafs.zip), it includes a setup.exe, just install it and youre ready to go!

Source Code

To build it you'll need "Microsoft Visual Studio Express 2013 for Windows Desktop" or better. At least thats what I used.

Using

  • On the first textbox write down the user/group that you want to add/alter/remove ACLs.
  • On the second textbox type in the folder that you wish to apply the ACL (allows folder starting by \\), or click Browse to use a SelectFolderDialog.
  • The current folder ACLs will be shown, change them as needed, and then click Apply.
  • Dont forget to check the recursive option if you want it to be recursive.
